## Design Specifications

### Color Palette
```
Primary:    #00A9CE (Macquarie Teal)
Secondary:  #003C71 (Navy Blue)
Accent:     #00D1A0 (Mint Green)
Text:       #1A1A1A (Dark Gray)
Background: #FFFFFF (White)
```

### Typography
- **Main Headline**: Macquarie Sans Bold, 72pt
- **Subheadline**: Macquarie Sans Regular, 48pt
- **Body Text**: Macquarie Sans Regular, 24pt
- **Small Text**: Macquarie Sans Regular, 18pt

### Visual Elements
- Robot emoji or custom TAAI mascot illustration
- QR code (minimum 3cm x 3cm)
- Macquarie logo (top, 5cm width)
- Directional arrows or icons
- Clean, modern design with ample whitespace

## Physical Poster Checklist

### Pre-Production
- [ ] Get brand approval from Corporate Communications
- [ ] Verify QR code functionality
- [ ] Test QR code at minimum scanning distance (30cm)
- [ ] Proof colors in CMYK
- [ ] Check legibility from 2-3 meters

### Production
- [ ] Print test copy first
- [ ] Verify colors match brand guidelines
- [ ] Check QR code scannability
- [ ] Laminate for durability
- [ ] Cut to size accurately

### Installation
- [ ] Measure poster locations
- [ ] Use appropriate mounting (tape, frame, stand)
- [ ] Ensure good lighting
- [ ] Position at eye level (150-170cm)
- [ ] Avoid glare on QR code

### Post-Installation
- [ ] Photo document all locations
- [ ] Test QR codes from actual viewing distance
- [ ] Monitor scan analytics
- [ ] Schedule regular checks (weekly)
- [ ] Plan refresh every 3-6 months

---

## Accessibility Considerations

1. **Text Contrast**: Ensure minimum 4.5:1 contrast ratio
2. **Font Size**: Minimum 24pt for body text
3. **QR Code Alternative**: Include text URL
4. **Placement Height**: 120-170cm for wheelchair accessibility
5. **Braille Option**: Consider tactile version for main locations

## Cost Estimate

### Per Location Setup
- **Poster Printing (A2, Full Color)**: $5-10
- **Lamination**: $3-5
- **Mounting Frame/Stand**: $15-30
- **Installation Labor**: $10-20
- **Total per location**: ~$40-60

### Organization-Wide Rollout
- **Locations**: 50 posters (Sydney, Melbourne, Brisbane)
- **Total Cost**: $2,000-3,000
- **Refresh (Quarterly)**: $500-1,000
